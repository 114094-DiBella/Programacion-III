1.Explicar y diagramar la arquitectura “Cliente-Servidor”

La arquitectura cliente-servidor es un modelo de arquitectura de software en el que un sistema se divide en dos partes principales: cliente y servidor. Este modelo es comun en servidores web, aplicaciones de red y servicios distribuidos.
El cliente envia solicitudes al servidor para obtener datos o realizar operaciones El servidor responde a las solicitudes de los clientes proporcionando los datos o realizando operaciones solicitadas
